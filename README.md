# Library_Management
#  Library Management System (Java, OOP + DSA)

This is a **console-based Java project** that demonstrates key **Object-Oriented Programming (OOP)** and **Data Structures (DSA)** concepts like **Linked List** and **Queue** through a simple library system.

---

##  Features
-  Add new books to the library  
-  Display all available books  
-  Search for a book by ID or title  
-  Issue and return books  
-  Maintain a queue of users waiting for issued books  
-  Remove books from the library  

---

## Concepts Used
| Concept | Description |
|----------|--------------|
| **Linked List** | Used to store books dynamically |
| **Queue** | Used to manage waitlist of users |
| **Encapsulation** | Private book fields with public methods |
| **Polymorphism** | Overridable display behavior |
| **OOP Principles** | Classes and objects to model real-world entities |
